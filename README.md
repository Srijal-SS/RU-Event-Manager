# RU Event Manager
This app is for managing events organised in college

Event Manager
A web app for colleges to manage events and notify students based on their interests.
What it does
Students can browse upcoming events and join/leave them
When a new event is added, students get notified if the event matches their interests, preferred roles, or if a friend is joining
Admins can add and delete events from an admin panel
Each student has a profile where they set their interests and preferred roles — this is what drives the notifications
Tech used
React + Vite
Zustand (state management)
React Router
date-fns (date formatting)
localStorage (to simulate a database in the demo)
Firebase (for production — Auth, Firestore, Cloud Messaging)
